Inicio

Misión: Ofrecer soluciones innovadoras de instrumentación virtual para el monitoreo y control inteligente de procesos industriales y tecnológicos.

Visión: Ser referentes en Latinoamérica en el desarrollo de sistemas de adquisición de datos y visualización virtual eficiente.

Este proyecto propone un sistema para monitorear condiciones críticas (temperatura, humedad, consumo energético e incendios) en centros de datos, utilizando sensores, Arduino y LabVIEW.

🖼️ Portafolio de Servicios

Problema: Monitorear condiciones críticas en centros de datos para prevenir fallos y optimizar recursos.

Metodología: Análisis del problema → Selección de sensores → Simulación en Proteus → Visualización y control en LabVIEW → Exportación de datos.

  • Fase 1: Identificación de variables y sensores
  • Fase 2: Implementación del sistema de adquisición en Proteus
  • Fase 3: Comunicación y visualización con LabVIEW
  • Fase 4: Control automático, almacenamiento y exportación de datos

❓ Preguntas Frecuentes

  • ¿Qué sensores se usaron? DHT11, ACS712 y sensor de llama.
  • ¿Qué software usaron? Proteus 8, LabVIEW, Arduino IDE.
  • ¿Cómo se comunica el sistema? A través de puertos seriales virtuales COM.
  • ¿Puedo usar este sistema en la vida real? Sí, con ajustes mínimos.
